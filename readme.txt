Inside the programs folder.

There are two programs:

The input file is samurai.raw

main.c - this generates sobelGx33,sobelGy33,sobelGx55,sobelGy55,Gradient3x3,Gradient5x5 raw images.

edge.c - this generates edge3x3,edge5x5 raw images.

To compile the program:

using the command prompt type make main

To run hit ./main

then compile the edge.c using make edge

To run hit ./edge

There will be a total of 9 raw images including the input image. 

The images folder contains images in jpg format.
